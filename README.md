# Todo List

## System Requirements
- PHP **V8**
- Composer **^V2**

## Installation

```
git clone https://github.com/ezehkingsleyuchenna/todo-backend.git
composer install
```

**Create .env file on the root folder**

**Copy env data from .env.example**

```
php artisan key:generate
```

**Create database.sqlite in database directory**

```
php artisan migrate
```
